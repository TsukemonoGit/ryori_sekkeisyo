# 第一部：【原理編】継続のための思考アルゴリズム

## 1. 操作（4つのみ）

| 操作 | 内容 | 使用条件 | 曜日 |
| :--- | :--- | :--- | :--- |
| 🔥 **焼く** | フライパンで放置加熱 | 時間10分以内 | 月・木・土 |
| 🍳 **炒める** | 強火で短時間 | 野菜あり | 水・金 |
| 🍝 **茹でて和える** | 麺＋油＋水 | 主食が麺 | 火 |
| 🍲 **煮る** | 全投入して加熱 | 余り2種類以上 | 日 |

### 操作の本質（科学原理）

| 曜日 | 操作原理（本質） | 具体的なアクション |
| :--- | :--- | :--- |
| **月** | **下味固定原理** | 下味冷凍→焼いてタレを絡めるだけで主菜を確定させる |
| **火** | **乳化・抽出原理** | 油＋にんにく＋麺＋茹で汁で味を成立させる |
| **水** | **強火・浸透圧原理** | 肉→野菜の順で強火短時間加熱し、塩で締める |
| **木** | **蒸気・脂質原理** | 冷凍魚をそのまま蒸し焼きし、脂とバターで完結させる |
| **金** | **粒状化原理** | ひき肉＋調味料を加熱前に混ぜてから加熱し、粒状にする |
| **土** | **乾燥・凝縮原理** | 味付き干物を弱火で焼き切るだけで完成させる |
| **日** | **集約・再構成原理** | 余り食材を一鍋に集約してカレー等に統合・再構成する |

## 2. 判断ルール（固定）

```
それ以外 → 麺（火曜）
```

## 3. 操作手順（そのまま実行）

### 🔥 焼く（12分）

**基本手順**
```
① フライパンに油 小さじ1（またはバター5g）
② 中火で30秒加熱
③ アルミホイルを敷く（くしゃっと丸めてから広げる）
④ 食材を置く（凍ったままで可）
⑤ フタをする
⑥ 中火4分
⑦ 裏返す
⑧ 弱火6分
⑨ 火を止めて2分放置（余熱）
```

**味付け**
- 肉 → 甘辛 または 塩
- 魚 → 塩 または 醤油 小さじ1

**確認ポイント**
- 押して固い → 完了
- 赤い汁 → 追加で2分加熱

---

### 🍳 炒める（8分）

**基本手順**
```
① フライパンに油 小さじ1
② 中火で肉を3分炒める
③ 野菜を入れる
④ 強火で2分
⑤ 塩 小さじ1/2（または甘辛タレ）
```

**ポイント**
- 肉→野菜の順番厳守
- 野菜は入れすぎない（100-150g）
- もやしは水が出るので強火短時間

---

### 🍝 茹でて和える（15分）

**基本：野菜入りペペロンチーノ**

| 材料 | 分量 |
|:---|:---|
| パスタ（または冷凍うどん） | 100g / 1玉 |
| キャベツ or もやし | 100〜150g |
| にんにく（チューブ） | 2cm |
| 鷹の爪（輪切り） | 少々 |
| オリーブオイル | 大さじ2 |
| ベーコン | 2〜3枚 |
| 塩（茹で湯用） | 小さじ2 |

**工程**
1. 大きめの鍋に湯を沸かし、塩を入れてパスタを袋の表示通りに茹でる
2. 茹でている間に：フライパンにオリーブオイル＋にんにく＋鷹の爪を入れ、**弱火で炒める**
3. にんにくが薄く色づいたら、ベーコンを加えて中火で1分炒める
4. **野菜を加えて強火で1分炒める**
5. **パスタの茹で汁大さじ2を加えて混ぜる**（乳化）
6. 茹で上がったパスタを入れて全体を和える
7. 塩で味を整えて完成

**ポイント**
- にんにくは弱火厳守（焦げると苦い）
- 茹で汁は必ず入れる（ソースが絡む）
- 野菜は麺と同量以下

---

### 🍲 煮る（15分）

**基本：一掃カレー**

| 材料 | 分量 |
|:---|:---|
| 余り食材（鶏肉・ベーコン・ひき肉・野菜端数等） | 適量 |
| 水 | 500ml |
| カレールー | 1/4箱（1かけ） |

**工程**
1. 鍋に余り食材を入れる
2. 水500mlを加える
3. カレールーを加える
4. 中火で15分煮込む

**具パターン**
- **ベーシック**：鶏肉or豚肉 + 野菜端数
- **ベーコン＆ひき肉mix**：ベーコン + ひき肉 + 野菜
- **全部入り**：鶏肉 + ベーコン + ひき肉 + 野菜（週末豪華版）

## 【月曜日】🍗 鶏肉の甘辛煮

### 当日の工程（15分）

**【解凍】**
- 朝：冷凍の鶏肉を冷蔵庫に移動（またはレンジ解凍3分）

**【調理】**
1. フライパン中火4分 → ひっくり返して弱火5分

**【タレ】**
| 材料 | 分量 |
|:---|:---|

| 砂糖 | 小さじ1 |

**工程**：肉を焼く → 余熱中にタレを煮詰める → かける

**【鶏もも照り焼き】**
- 皮目から焼く（油が出る）
- タレを入れて煮絡める
- ジューシーに仕上がる

**【手羽先甘辛煮】**
- 下味なしでOK
- タレ＋水大3を入れて15分煮込む

**【砂肝塩炒め】**
- 塩胡椒でサッと炒め
- 歯ごたえ重視

**【鶏ひきつくね風】**

## 【水曜日】🥓 豚肉の炒め物

### 前日準備
- **なし**（ただし木曜の魚を冷凍庫左手前に移動しておく）

### 基本：豚こま塩ダレ炒め（8分）

| 材料 | 分量 |
|:---|:---|
| 豚こま切れ肉 | 200g |
| キャベツ（または今週の野菜） | 2把（150g） |
| 塩 | 小さじ1/2 |
| 酒 | 大さじ1 |
| ごま油 | 小さじ1 |
| にんにく（チューブ） | 1cm（あれば） |

**工程**
1. 野菜を食べやすい大きさにカット
2. フライパンにごま油を熱し、豚肉を炒める（中火3分）
3. 肉に火が通ったら野菜を入れる
4. 塩・酒を加えて強火で2分炒める
5. 完成

### 代替パターン

**【豚バラ炒め】**

## 【木曜日】🐡 白身魚のバター焼き

### 前日準備（水曜日の夜に）
- **冷凍庫の左手前に鮭（またはタラ等）を移動しておく**

### 基本工程（12分）

1. フライパンにバター5g（鮭は3g）
2. **アルミホイルを敷く**（くしゃっと丸めてから広げると焦げ付きにくい）

4. **蓋をして中火4分**
5. ひっくり返して醤油小さじ1を回し入れる
6. **蓋をして弱火で完焼**（残り8分）

| ブリ（冷凍） | 10分 | 照り焼き・塩焼きもOK |

**【タラのムニエル風】**

- 仕上げにレモン汁（あれば）

**【カレイの煮付け】**
- バター不要

## 【土曜日】🐠 干物の炭火風

### 前日準備（金曜日の夜に）
- **冷凍庫の左手前にサバみりん干し（またはホッケ等）を移動**

### 基本：サバみりん干し（10分）

| 材料 | 分量 |
|:---|:---|
| サバみりん干し（冷凍） | 1枚 |
| （バター不要） | |

**工程**
1. フライパンを中火で温める
2. **アルミホイルを敷く**
3. 凍ったまま干物を入れる（**身の部分を下**に）
4. **蓋をして弱火で5〜7分**（身側を蒸し焼き）
5. **裏返して皮側を2〜3分**（焦げ目がついたらOK）

**ポイント**
- みりん干しは**味付け済み**なので調味料不要

- 焼きすぎるとパサつくので、「少し生かも？」と思うくらいがベスト

### 焼き時間早見表

| 魚 | 時間 | ポイント |
|:---|:---|:---|

| ホッケ開き | 12分 | 身が厚いので12分。バター風味も合う |
| カマス | 10分 | 塩焼きでシンプルに。塩を軽くふる |
| アジ開き | 10分 | 干物焼き or 南蛮漬け風 |

### 代替パターン

**【ホッケ開きのバター焼き】**
- アルミホイル＋バター5g
- 12分じっくり焼く

**【カマスの塩焼き】**
- アルミホイルを敷く
- カマスに塩を軽くふって焼く（10分）

**【アジ開きの南蛮漬け風】**
- 干物焼きで焼く（10分）
